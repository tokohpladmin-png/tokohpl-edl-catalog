# V13 Exclude EDL ABS Edging

Updated Zoho product filtering so EDL ABS Edging items are not shown on the website.

Excluded if item name/SKU/brand/description contains:
- ABS EDGING
- ABS EDGE
- EDGING
- W23MM
- T1.0MM
- 23*1 / 23 x 1 / 23x1

Example excluded item:
`DSS 00075SM 23*1 - EDL ABS EDGING W23MM X T1.0MM | APTICO DSS 00075SM 23 x 1 MM`
